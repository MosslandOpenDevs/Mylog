## 🤔 고민했던 것들

### 자산 위젯 어떤 것을 넣을지
- 처음에는 부동산 + 주식 + 환율 3개만 고려
- 예적금, 코인, 차량, 대출, 연금, 청약 캘린더까지 확장
- **유료 API 필요 여부** 확인 → 전부 무료 또는 사용자 수동 입력으로 해결 가능

### 대시보드 레이아웃 구조
- 우측 사이드바를 **자산 전용**으로 쓸지, 쇼핑/취미도 섞을지 고민
- 결론: **우측은 자산 Only**, 쇼핑/취미는 좌측 피드 영역에 배치

### 좌측 피드 카테고리
- 아직 최종 확정 안 됨 (사용자가 "아직 생각중"이라고 언급)
- 현재는 더미 데이터로 3섹션 채워둔 상태
- Phase 2에서 실제 RSS / API 연동 시 재구성 필요

### CSS 구조
- 인라인 스타일 vs CSS 클래스 → **CSS 클래스로 통일** 결정
- 위젯마다 제각각이던 정렬 기준을 `widget-row`, `widget-block`, `widget-list` 패턴으로 표준화

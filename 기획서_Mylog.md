# 📌 Mylog — 나만의 개인화 포털사이트

> **"로그인하면, 내 세상이 펼쳐진다."**
>
> Mylog는 네이버·다음 같은 포털사이트의 프레임에,
> 오직 **'나'에게 최적화된 정보**만 보여주는 **개인 전용 포털**입니다.

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Mylog (마이로그) |
| **한줄 소개** | 나의 관심사·취미·루틴·위시리스트를 한눈에 보는 개인화 포털 |
| **플랫폼** | 웹 (반응형, 모바일 대응) |
| **타겟 사용자** | 자기 관리에 관심 있는 20~40대 |
| **핵심 가치** | 개인화 · 한눈에 보기 · 생산성 향상 |

---

## 2. 핵심 컨셉

### 🎯 문제 정의
- 기존 포털(네이버, 다음)은 **모든 사람에게 동일한 정보**를 보여줌
- 내 관심사, 취미, 루틴, 위시리스트 등이 여러 앱/서비스에 **분산**되어 있음
- 매일 아침 여러 앱을 돌아다니며 정보를 확인하는 **비효율**

### 💡 해결책
- 로그인하면 **나만을 위한 대시보드**가 펼쳐지는 포털
- 관심사, 취미, 루틴, 위시리스트 등을 **위젯 형태**로 자유롭게 배치
- 하나의 화면에서 **내 하루를 시작**할 수 있는 원스톱 서비스

## 3. 주요 기능

### 3.1 🔐 로그인 / 회원가입
| 기능 | 설명 |
|------|------|
| 소셜 로그인 | Google, Kakao, Naver 등 간편 로그인 |
| 프로필 설정 | 닉네임, 프로필 사진, 관심사 태그 초기 설정 |
| 온보딩 | 첫 로그인 시 관심사·취미 선택 가이드 |

---

### 3.2 🏠 메인 대시보드 (홈)
> 로그인 후 가장 먼저 보이는 화면. 포털의 핵심.

#### 위젯 시스템
사용자가 원하는 위젯을 **드래그 앤 드롭**으로 자유롭게 배치

| 위젯 | 설명 |
|------|------|
| ☀️ **모닝 루틴** | 아침에 해야 할 일 체크리스트 (기상, 운동, 명상 등) |
| 📰 **관심 뉴스** | 설정한 관심사 기반 뉴스 피드 |
| 🎮 **취미 트래커** | 취미 활동 기록 및 진행 상황 |
| 🛒 **위시리스트** | 갖고 싶은 것들 목록 (가격 추적 포함) |
| 📅 **오늘의 일정** | 캘린더 연동 일정 표시 |
| 🌤️ **날씨** | 현재 위치 기반 날씨 정보 |
| ✅ **투두리스트** | 오늘 할 일 관리 |
| 📊 **목표 달성률** | 주간/월간 목표 진행 상황 |
| 🔖 **북마크** | 즐겨찾기 사이트 바로가기 |
| 💰 **가계부 요약** | 이번 달 수입/지출 요약 |
| 🏠 **부동산** | 관심 지역 시세, 매물 알림, 청약 일정 |
| 📈 **주식 / 투자** | 관심 종목 시세, 포트폴리오 수익률, 경제 뉴스 |
| 🚗 **자동차** | 관심 차량 시세, 신차 출시 정보, 유지비 관리 |
| 💪 **건강 / 운동** | 운동 기록, 걸음 수, 식단 메모, 건강 체크 |
| ✈️ **여행 / 맛집** | 가고 싶은 여행지, 맛집 리스트, 항공권 가격 추적 |
| 🏷️ **쇼핑 핫딜** | 관심 카테고리 최저가 알림, 타임딜 모아보기 |

## 5. 디자인 방향

### 🎨 디자인 원칙
| 원칙 | 설명 |
|------|------|
| **미니멀 & 클린** | 정보 과부하를 줄이고, 핵심만 보여주는 깔끔한 디자인 |
| **다크모드 기본** | 세련된 느낌의 다크 테마 기본, 라이트 모드 전환 가능 |
| **카드 기반 UI** | 위젯을 카드 형태로 구성, 직관적 정보 소비 |
| **부드러운 애니메이션** | 페이지 전환, 위젯 인터랙션에 미세 애니메이션 적용 |

### 🖼️ 레이아웃 참고
- **헤더**: 로고 + 검색바 + 알림 + 프로필 (네이버/다음 스타일)
- **사이드바**: 주요 메뉴 네비게이션 (접기 가능)
- **메인 영역**: 위젯 그리드 (반응형, 드래그 앤 드롭)
- **푸터**: 간단한 링크, 저작권 정보

### 📍 Phase 2 — 핵심 기능
> 목표: 개인화 포털의 핵심 가치 완성

- [ ] 위젯 드래그 앤 드롭 배치

> 목표: 진짜 '내 포털'이 되는 경험 완성

- [ ] 관심사 기반 콘텐츠 큐레이션
- [ ] 주간/월간 마이 리포트
- [ ] 목표 설정 및 성장 그래프

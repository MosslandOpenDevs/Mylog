# 📌 Mylog — 나만의 3대 관심사 최적화 대시보드

> **"지갑이 열리고, 시간이 쓰이고, 돈이 모이는 곳."**
>
> Mylog는 세상의 모든 정보가 아닌, 직장인의 핵심 관심사인
> **'쇼핑(Shopping)', '취미(Hobbies)', '자산(Assets)'** 단 3가지만을 추적하고 관리하는 **초개인화 포털**입니다.

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Mylog (마이로그) |
| **한줄 소개** | 나의 쇼핑 위시리스트, 취미 생활, 그리고 자산의 흐름을 한눈에 보는 개인화 대시보드 |
| **플랫폼** | 웹 (반응형, 모바일 대응) |

| **핵심 가치** | 선택과 집중 · 한눈에 보기 · 비용 없는 효율성 |

## 2. 핵심 컨셉

### 🎯 문제 정의
- 기존 포털(네이버, 다음)은 **모든 사람에게 동일한 정보**를 보여줌
- 내 관심사, 취미, 위시리스트 등이 여러 앱/서비스에 **분산**되어 있음
- 매일 아침 여러 앱을 돌아다니며 정보를 확인하는 **비효율**

### 💡 해결책
- 로그인하면 **나만을 위한 대시보드**가 펼쳐지는 포털
- 쇼핑, 취미, 자산 데이터를 **위젯 형태**로 자유롭게 배치
- 하나의 화면에서 **내 하루를 시작**할 수 있는 원스톱 서비스

---

### ⚖️ 일반 포털과의 3가지 핵심 차이점
가장 큰 차이점은 **"불필요한 정보의 차단(노이즈 캔슬링)"**과 **"생산성 도구(앱)들의 통합"**입니다. 일반 포털이 '모두를 위한 백화점'이라면, Mylog는 철저히 **'나 한 사람만을 위한 비서의 요약 보고서'** 역할을 합니다.

1. **원치 않는 뉴스나 광고가 0% (진정한 큐레이션)**
   - **일반 포털**: 접속하자마자 내가 관심 없는 정치 싸움, 자극적인 연예 가십, 팝업 광고가 시선을 뺏습니다.
   - **Mylog**: 오직 내가 구독한 키워드만 노출됩니다.
   
2. **포털 + 생산성 도구의 결합 (앱 통합)**
   - **일반 포털**: 정보 검색과 뉴스 소비에 그칩니다.
   - **Mylog**: 뉴스를 보는 동시에, 자산 현황, 위시리스트 최저가 추적 등을 한 화면에서 모두 관리합니다.
   
3. **위젯 기반의 자유로운 커스터마이징 (UI의 주도권)**
   - **일반 포털**: 정해진 레이아웃을 수동적으로 봐야 합니다.
   - **Mylog**: 내 현재 목표에 따라 화면을 자유롭게 재배치합니다.

## 3. 주요 기능

### 3.1 🔐 로그인 / 회원가입
| 기능 | 설명 |
|------|------|
| 소셜 로그인 | Google, Kakao, Naver 등 간편 로그인 |
| 프로필 설정 | 닉네임, 프로필 사진, 관심사 태그 초기 설정 |
| 온보딩 | 첫 로그인 시 관심사·취미 선택 가이드 |

### 3.2 🏠 메인 대시보드 (홈)
> 로그인 후 가장 먼저 보이는 화면. **쇼핑, 취미, 자산** 데이터의 요약판.

#### 좌측: 콘텐츠 피드 영역
| 피드 섹션 | 설명 | 무료 데이터 활용 방안 |
|-----------|------|----------------------|
| 🛒 **실시간 핫딜 보드** | 커뮤니티 핫딜 정보를 카테고리별 필터링 | 유명 커뮤니티 핫딜 게시판 RSS 피드 |
| � **덕질 뉴스 피드** | IT/테크, 콘솔 게임, 운동 등 관심사별 뉴스 | 언론사/블로그 RSS + YouTube Data API |
| 💰 **자산 & 투자 모니터링** | 관심 부동산, 주식, 가상화폐 관련 뉴스 | 국토부 실거래가 API + 금융 RSS |

#### 우측: 자산 사이드바 (위젯 10개)
| # | 위젯 | 설명 | 데이터 소스 |
|---|------|------|------------|
| ① | 📊 **총 자산 요약** | 순자산 + 자산 비중 바 (부동산/주식/예적금/코인/차량) | 자체 계산 (사용자 입력 기반) |
| ② | 🏠 **내 부동산** | 내 아파트 시세, 매입가 대비 상승분 | 국토부 실거래가 API (무료) |
| ③ | 📈 **내 주식** | 보유 종목, 현재가, 수익률, 총 평가금액 | 사용자 수동 입력 + 무료 시세 API |
| ④ | � **내 예적금** | 은행 예금/적금 잔고, 만기일, 예상 이자 | 사용자 수동 입력 |
| ⑤ | 🪙 **내 코인** | BTC, ETH 등 보유 가상화폐 현황 | 업비트 Open API (무료) |
| ⑥ | � **내 차** | 차량 모델, 중고 시세 추정, 감가 추적 | 사용자 수동 입력 |
| ⑦ | 💳 **내 대출** | 주담대/신용대출 잔액, 다음 상환일, 월 상환 합계 | 사용자 수동 입력 |
| ⑧ | � **연금/퇴직금** | 국민연금 예상 수령액, 퇴직연금 적립 현황 | 사용자 수동 입력 |
| ⑨ | 📅 **청약 캘린더** | 이번 주/달 아파트 청약 일정 D-day | 한국부동산원 청약홈 API (무료) |
| ⑩ | 💱 **환율** | 원/달러, 원/엔, 원/유로 컴팩트 표시 | 한국수출입은행 API (무료) |

### 3.3 🛒 쇼핑 (Shopping) 딥 다이브

|------|------|

| 관심 부동산 모니터링 | 지정한 아파트 단지의 가장 최근 "실거래가"만 깔끔하게 리스팅 (무료 API) |
| 거시 경제 지표판 | 자산 관리에 필수적인 오늘의 주요 환율 및 금리 스냅샷 제공 |

## 4. 화면 구성 (IA: Information Architecture)

```
Mylog
├── 🏠 홈 (메인 대시보드)
│   ├── 좌측: 콘텐츠 피드 (쇼핑 핫딜 / 취미 뉴스 / 자산 모니터링)
│   ├── 우측: 자산 사이드바 (자산 위젯 10개)
│   └── 알림 센터 (가격 알림, 핫딜, 청약 알림 등)
│
├── 🛒 쇼핑 탭
│   ├── 위시리스트 & 가격 추적
│   ├── 나의 지출/예산 요약
│   └── 실시간 핫딜 모아보기
│
├── 🎮 취미 탭
│   ├── 취미 활동 기록 (잔디심기)
│   ├── 취미 아카이브 (리스트/별점)
│   └── 덕질 피드 (RSS / 유튜브 큐레이션)
│
├── 💰 자산 탭
│   ├── 내 자산 파이 차트 & 흐름
│   ├── 실시간 환율 / 금리 지표
│   └── 부동산 실거래가 / 청약 캘린더
│
└── ⚙️ 설정
    ├── 프로필 / 관심 키워드 관리
    ├── 위젯 에디터 (배치/추가/삭제)
    └── 데이터 관리 (로컬/클라우드 동기화)
```
